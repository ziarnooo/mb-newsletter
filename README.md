## How to download

Click any file above to view it, then use the "Download raw file" button (top-right).

Or grab everything as a ZIP: [Download ZIP](https://github.com/ziarnooo/mb-newsletter/archive/refs/heads/main.zip)

## How to use

The `index.html` references images via raw.githubusercontent.com URLs, so it will render correctly when sent as an email - no extra setup required.
